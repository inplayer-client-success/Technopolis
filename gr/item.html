Πρόγραμμα

Χρειάζεστε βοήθεια;

  • Σύνδεση και πληρωμή

    Πώς δημιουργώ λογαριασμό;

    Βήμα 1.Κάντε κλικ στο κουμπί Login ή στο περιεχόμενο που θέλετε να παρακολουθήσετε.

    Βήμα 2.Επιλέξτε Sign up στο επάνω μέρος της φόρμας.

    Βήμα 3. Συμπληρώστε τη φόρμα εγγραφής και κάντε κλικ στο κουμπί Register

    Θα λάβετε μια επιβεβαίωση μέσω e-mail για την επιτυχή δημιουργία του λογαριασμού σας. Για να μάθετε σχετικά με το πώς μπορείτε να κάνετε μια αγορά, συνεχίστε στην επόμενη ενότητα.

    Πώς μπορώ να κάνω μια αγορά;

    Βήμα 1. Κάντε κλικ στο περιεχόμενο που θέλετε να δείτε.

    Βήμα 2. Εισάγετε τα διαπιστευτήριά σας (διεύθυνση e-mail και κωδικό πρόσβασης).

    Βήμα 3. Επιλέξτε την επιθυμητή επιλογή πληρωμής και πατήστεNext.

    Βήμα 4. Εισάγετε τις πληροφορίες που απαιτούνται.

    Βήμα 5.Κάντε κλικ στο κουμπί Pay

    Θα λάβετε αμέσως πρόσβαση και ένα μήνυμα e-mail που επιβεβαιώνει την αγορά σας. Σε περίπτωση που δεν φορτώσει η σελίδα, πατήστε το κουμπί Ανανέωση.

    Πώς μπορώ να συνδεθώ και να βλέπω;

    Βήμα 1.Πατήστε το κουμπί Login ή κάντε κλικ στο περιεχόμενο που θέλετε να δείτε.

    Βήμα 2. Στη φόρμα που εμφανίζεται, εισαγάγετε τα διαπιστευτήριά σας (διεύθυνση e-mail και κωδικό πρόσβασης).

    Βήμα 3.Πατήστε το κουμπί Login.

    Έπειτα, η σελίδα θα ανανεωθεί και εάν έχετε ήδη κάνει αγορά, θα σταματήσει να εμφανίζεται το συνδρομητικό τείχος. Εάν δεν έχετε κάνει αγορά, ακολουθήστε τις οδηγίες παραπάνω στην ενότηταΠώς μπορώ να κάνω μια αγορά.

    Πώς κάνω επαναφορά του κωδικού μου πρόσβασης;

    Βήμα 1.Πλοηγηθείτε στη φόρμα σύνδεσης κάνοντας κλικ στο κουμπί Loginή στο περιεχόμενο που θέλετε να δείτε.

    Βήμα 2. Πατήστε τον σύνδεσμοForgot Passwordκάτω από το πεδίο του κωδικού πρόσβασης στη φόρμα σύνδεσης.

    Βήμα 3.Εισάγετε τη διεύθυνση e-mail σας και πατήστε το κουμπί Reset

    Βήμα 4. Θα εμφανιστεί ένα νέο παράθυρο. Βεβαιωθείτε ότι θα το αφήσετε ανοικτό.

    Βήμα 5. Ελέγξτε τη διεύθυνση e-mail σας. Θα λάβετε έναpassword reset token. Εάν δεν το δείτε στα εισερχόμενά σας, βεβαιωθείτε ότι ελέγξατε τους φακέλους ανεπιθύμητης αλληλογραφίας και κάδου απορριμμάτων.

    Βήμα 6. Κάντε αντιγραφή του Tokenκαι επικόλληση στο πρώτο πεδίο του παραθύρου που εμφανίστηκε προηγουμένως.

    Βήμα 7. Εισάγετε τον νέο κωδικό σας πρόσβασης στο δεύτερο και το τρίτο πεδίο.

    Βήμα 8. Πατήστε Set

    Εάν δεν λάβετε διακριτικό επαναφοράς κωδικού πρόσβασης, επικοινωνήστε μαζί μας στη διεύθυνση support@inplayer.com.

    Μπορώ με τον λογαριασμό μου να βλέπω σε πολλές συσκευές;

    Όχι. Μπορείτε να παρακολουθήσετε σε μία συσκευή χρησιμοποιώντας τον ίδιο λογαριασμό.
